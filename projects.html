Project:1 - Multi Purpose Robot.

Multi Purpose Robot
Project Features
  • Object Following Robot
  • Obstacle Avoiding Robot
  • Bluetooth Control Robot
  • Voice Command Robot
Project Keywords
  • Arduino Uno
  • Motor Driver Shield
  • Bluetooth Module
  • Ultrasonic and IR Sensors
  • Servo and TT Gear Motors

Project Description

The Multi-Purpose robot or Creating an object-following, obstacle-avoiding, and Bluetooth-controlled robot involves combining Arduino, a motor driver shield (such as L293D), and various sensors for enhanced functionality. The Arduino serves as the brain, processing signals and controlling the robot’s movements. The L293D motor driver shield interfaces with the Arduino, allowing precise control of the DC motors that propel the robot. For object following, infrared or ultrasonic sensors detect and track objects. Obstacle avoidance involves programming the robot to navigate around obstacles using sensor data. Integrating Bluetooth technology enables wireless control through a mobile app, adding a remote dimension to the robot’s operation. By arduino developed apps it also possible by voice command through google assistance for controlling the robot car as the set command. This multifunctional robotic platform demonstrates the versatility of Arduino and motor driver shields in creating intelligent, interactive machines.

Project:3 - Weather Scout.

Weather Scout.
Project Features
  • Get Current Weather Info.
  • Get Weather Forecast.
Project Keywords.
  • HTML
  • CSS
  • Javascript DOM
  • API Fetch

Project Description

WeatherScout is a web application designed to provide users with current weather information for any city they search for. Utilizing the OpenWeatherMap API, the application fetches and displays data such as temperature, humidity, wind speed, pressure, and weather description. Users simply enter the name of a city and receive up-to-date weather details, making it a convenient tool for anyone needing quick and accurate weather updates. The interface is user-friendly, featuring a clean design with a background image that complements the weather theme. Additionally, the application includes an error handling feature that informs users if a city is not found, enhancing the overall user experience.

Project: 6 - Rock Paper Scissors Game

Rock Paper Scissors Game
Project Features
  • Simple Rock Paper Scissors Game
  • Account Login & Creating System by Firebase
  • Save Game Scores to Account
Project Keywords.
  • HTML
  • CSS
  • Javascript DOM
  • Firebase

Project Description

A web-based Rock, Paper, Scissors game integrated with Firebase for authentication and real-time score tracking. Players can log in using Google to create an account and save their game scores to the Firebase database, which enables seamless score synchronization across devices. The sign-in and sign-out features ensure a secure and personalized experience. Players can log out anytime, and upon logging back in, their game history and progress are restored, allowing them to continue where they left off.
